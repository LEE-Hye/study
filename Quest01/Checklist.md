> ## 2. 브라우저의 역사는 어떻게 될까요?

<br>

* 브라우저란❓

    - 웹 페이지, 이미지 등의 콘텐츠를 수신, 전송 및 표현하는 소프트웨어

<br>

|년도|브라우저|설명|
|----|-------------|---|
|1991|월드 와이드 웹|세계 최초 웹 브라우저|
|1993|모자이크|최초로 그래픽 기반의 웹 브라우저인 텍스트와 이미지를 함께 보여줌|
|1994|넷스케이프 내비게이터, 넷스케이프 커뮤니케이터|넷스케이프 회사가 개발안 웹 브라우저로 모자이크 웹 브라우저를 개발했던 마크 앤드리슨이 개발 / 초기 인터넷 사용자들의 폭발적 반응을 얻어 90% 시장 점유율 차지 /후에 오픈 소스 웹 브라우저계에 공헌을 함
|1995|인터넷 익스플로러(IE)|모자이크 기반으로 익스플로어 개발|
|1차 인터넷 전쟁|넷스케이프 vs IE| 시장 점유율 90퍼 돌파/ 마이크로소프트 윈도우 운영체제에 익스플로러를 기본으로 설치하게 함 / 윈도우 운영체제에 8~90퍼 점유율이던 넷스케이프에게 윈도우용 개발을 하지말라고 함-> 싫다함/거래처들에게 윈도우를 주는 계약 조건으로 넷스케이프를 쓰지 못하게 하고 평생 무료이며 운영체제와 브라우저는 기술적으로 통합된 시스템이라고 하면서 1차 인터넷 브라우저 전쟁에서 넷스케이프가 지고 IE가 점유율 90퍼 찍음|
|1차 인터넷 전쟁|넷스케이프 vs IE|- 시장 점유율 전쟁 <br> - 각 브라우저가 다양한 기술을 적용 <br> - 브라우저 간 호환성 결여|
|1996|오페라|다양한 기능을 통합해서 기본적으로 탑재하고 있으면서 같은 종류의 다른 프로그램에 비해 작고, 렌더링 속도가 가볍고 빠름|
|2002|모질라 애플리케이션 스위트|기능과 안전성 면에서 기존 커뮤니케이터를 능가하는 제품만듦|
|2003|사파리|북마크 관리 체계를 가지고, 퀵타임 멀티미디어 기술과 통합하고, 탭 브라우징 인터페이스 사용, 구글 검색 상자 만듦|
|2004|모질라 파이어 폭스|넷스케이프가 미래의 웹 브라우저를 위해 오픈소스로 공개했는데 그걸 기반으로 천천히 개발 및 기능 추가 꾸준히 점유율을 높이고 있음/게코 레이아웃 엔진을 사용해 웹 표준 구현/사파리의 가벼운 웹 키트 엔진과 js를 더 빠르게 실행하기 위한 v8 엔진을 얹어 만듦|
|2008|구글 크롬|Google에서 개발 및 서비스 하는 웹 브라우저/WebKit에서 갈라져 나온 블링크 엔진|
|2차 브라우저 전쟁|IE vs 파이어 폭스 vs 크롬|과도한 플러그인과 부가기능으로 인해 브라우저들이 무거워져 다시 가벼운 상태로 돌아가려는 분위기에 크롬과 파이어 폭스는 업데이트를 시작하였고 이미 무거울대로 무거워진 IE는 업데이트가 힘들어 파이어폭스가 IE점유율을 뺏고 크롬이 점유율 1위를 함|
|2차 브라우저 전쟁|IE vs 파이어 폭스 vs 크롬|- 마이크로소프트 사 웹 브랄우저 개발에 소홀함 <br> - 다양한 웹 브라우저 등장 <br> - 무겁고 개발이 정체된 웹 브라우저에 빠르고 강력한 브라우저들 등장 |

>> ### 2-1. Internet Explorer가 브라우저 시장을 독점하면서 어떤 문제가 일어났고, 이 문제는 어떻게 해결되었을까요?

<br>

- 경쟁할 브라우저가 없으니 기술 발전이 이루어지지 않는다.
    
    - 최신 웹 표준의 지원과 버그 수정이 느림

    <br>

- 2차 브라우저 전쟁으로 IE가 몰락하게 됨

    - IE는 Active X등 다른 웹 브라우저가 사용하지 않는 기술을 사용하면서 웹 표준과 거리가 있는 길을 걸어감 -> 이거 보안에 취약함

    - 브라우저들이 다시 가벼운 상태로 돌아가려고 하고 있어 그에 맞춰 크롬과 파이어폭스는 업데이를 하며 새로운 버전을 내놓았는데 IE는 과도한 플러그인과 기능으로 무거워질대로 무거워져 다른 브라우저의 업데이트를 따라가지 못하여 크롬과 파이어폭스에게 점유율을 빼앗겼다.
